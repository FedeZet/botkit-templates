## ðŸš€ Getting Started

### Install Dependencies

```bash
npm install
```

### Configure Bot

Enter your bot's token, clientId, prefix and guildId in `.env`. You can find your token [here](https://discordapp.com/developers/applications/me).

```
# Discord Bot
DISCORD_CLIENT_TOKEN=[your token]
DISCORD_CLIENT_ID=[your client id]
PREFIX=[your prefix]

# Guild
DISCORD_GUILD_ID=[your guild id]
```

> **Note:** No brackets or quotes are needed.

### Deploy commands

```bash
npm run deploy
```

### Run bot

```bash
npm start
```
